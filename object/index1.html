陣列、物件 - 陣列方法(forEach、map、filter、find、every、some、reduce)

forEach

寫法 people.forEach(function (item, index) { ... };

範例

people.forEach(function (item, index) {  console.log(index,item);// 每一筆資料 });

map

寫法 const MapPeople =people.map(function (item, index) {
 return item ;
};

特性 1. map 會回傳值，以陣列呈現，如果沒有return，則陣列內的值會是 undefined 2. 不管結果是true 或 false，都會回傳一個值

範例

const MapPeople = people.map(function(item,index){  item.icash = item.money + 500;  console.log(item);// 每一筆資料  return item; }); console.log(MapPeople); //輸出
[
 {
  name: '小明',
  money: 500,
  icash: 1000
 },
 {
  name: '漂亮阿姨',
  money: 3000,
  icash: 3500
 },
 {
  name: '杰倫',
  money: 60000,
  icash: 60500
 }
]

filter

寫法 const FilterPeople =people.filter(function (item, index) {
  ...
};

特性 1. filter會回傳結果為true的值，以陣列呈現。

範例

const FilterPeople = people.filter(function(item,index){  if(item.money < 5000){   return true;  } }); console.log(FilterPeople); //輸出
[
 {
  name: '小明',
  money: 500,
 },
 {
  name: '漂亮阿姨',
  money: 3000,
 }
]

find

寫法 const findPeople =people.find(function (item, index) {
  ...
};

特性 1. find會回傳結果為true的值，只會回傳一次。

範例

const findPeople = people.find(function(item,index){  if(item.money < 5000){   return true;  } }); console.log(findPeople); //輸出
{
 name: '小明',
 money: 500,
}

every

寫法 const everyPeople =people.every(function (item, index) {
  ...
};

特性 1. every必須所有結果都為true，結果才會是true。

範例

const everyPeople = people.every(function(item,index){  if(item.money < 5000){   return true;  } }); console.log(everyPeople);// false

some

寫法 const somePeople =people.some(function (item, index) {
  ...
};

特性 1. some只要有一個為true，結果就會是true。

範例

const somePeople = people.some(function(item,index){  if(item.money < 5000){   return true;  } }); console.log(somePeople);// true

reduce

寫法 people.reduce(function(前一個值,當前值,索引){
 return prev + item.money;
}, 前一個值的初始值);

範例 - 從初始值0開始加總

const ReducePeople = people.reduce(function (prev, item, index) {  return prev + item.money; }, 0); console.log(ReducePeople);// 63500

範例 - 比大小

const ReducePeople = people.reduce(function (prev, item, index) {  return Math.max(prev,item.money); }, 0); console.log(ReducePeople);// 60000
