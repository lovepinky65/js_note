AJAX - AJAX基本概念

AJAX基本寫法

// 產生一個XMLHttpRequest var xhr = new XMLHttpRequest(); // .open('格式','網址',同步或非同步) xhr.open('get','https://hexschool.github.io/ajaxHomework/data.json',true); // 傳送資料過去，沒傳送用null xhr.send(null); // 看一下目前狀態 console.log(xhr.readyState);

readySate狀態說明 0 - 已經產生一個XMLHttpRequest()，但還沒有連結你要的資料
1 - 用了open()，但還沒把資料傳送過去
2 - 偵測有用send
3 - loading
4 - 撈到資料，數據已經完全接收到了
