localStorage瀏覽器資料儲存 - JSON.stringify()、JSON.parse()

JSON.stringify()、JSON.parse()

JSON.stringfiy() - 將Array轉string JSON.stringfiy(Array);

JSON.parse() - 將string轉Array JSON.parse(string)

程式碼有範例

var array = [ {  name : 'pinky' }]; // 將陣列轉換成字串
var chageArray = JSON.stringify(array); // 再把字串加入localStorage
localStorage.setItem('myArray',chageArray); // 取出localStorage值
var getArray = localStorage.getItem('myArray'); // 確認值的type
console.log(typeof(getArray)); // 將字串轉換成陣列
var chageString = JSON.parse(getArray); console.log(chageString); console.log(typeof(chageString));
