ES6 - 縮寫

物件縮寫

原本寫法

const TeamOne = '魯夫'; const TeamTwo = {  title : '喬巴',  content : '娜美' }; const TeamAll = {  TeamOne : TeamOne,  TeamTwo : TeamTwo };

新的寫法

const TeamOne = '魯夫'; const TeamTwo = {  title : '喬巴',  content : '娜美' }; const TeamAll = {  TeamOne,  TeamTwo };

物件函式

原本寫法

const newTeam = {  showPost : function(){   console.log('內容');  } }

新的寫法

const newTeam = {  showPost(){   console.log('內容');  } }

搭配解構使用

原本寫法(傳參考)

const orderTeam = {  showPost : {   name : '小名'  },  showTitle : '媽媽' }; const newTeam = orderTeam; newTeam.ming = '天天';

新的寫法(不傳參考)

const orderTeam = {  showPost : {   name : '小名'  },  showTitle : '媽媽' }; const newTeam = {  ...orderTeam }; newTeam.ming = '天天';
