ES6 - 輕鬆組字串相加

用``包起來，變數寫法為${變數名}

原本寫法

const name = 'Erica'; const dog = 2; let contentbox = document.querySelector('.js_content'); contentbox.innerHTML = '<ul><li>'+name+'有'+dog+'隻嗷嗷</li></ul>';

新的寫法

const name2 = 'Pinky'; const dog2 = 1; let contentbox2 = document.querySelector('.js_content2'); contentbox2.innerHTML = `  <ul>   <li>${name2}有${dog2}隻嗷嗷</li>  </ul> `;
