ES6 - 展開與其餘

合併陣列的方法

原本寫法

let groupA=['小明' , '媽媽' , '爸爸']; let groupB=['大名' , '阿姨']; let groupAll = groupA.concat(groupB);

新的寫法

let groupA=['小明' , '媽媽' , '爸爸']; let groupB=['大名' , '阿姨']; let groupAll = [...groupA, ...groupB];

淺複製

原本寫法(傳參考)

let groupC = ['小明', '媽媽', '爸爸']; let groupD = groupC; groupD.push('小黑');

新的寫法(淺複製，沒傳參考)

let groupC = ['小明', '媽媽', '爸爸']; //新陣列 let groupD = [...groupC]; groupD.push('小黑');

let doms = document.querySelectorAll('li'); let domsArray = [...doms];

類陣列 - function 傳入值

function 傳入值為類陣列(arguments)

function updateEasy() {  //arg 為類陣列  let arg = arguments;  console.log(arguments); } updateEasy(2,3,5,100,200,60);

將 arguments 展開成陣列

function updateEasy() {  //arg 為陣列  let arg = [...arguments];  console.log(arguments); } updateEasy(2,3,5,100,200,60);

其餘參數

下方寫法，只能傳入固定的參數

function moreMoney(a,b,c){  ... } moreMoney(100,200,300);

在無法確定傳入參數的多寡時，可以用其餘參數的方式傳入 會以陣列方式呈現

function moreMoney(...money){  console.log(money);//[100, 200, 300] } moreMoney(100,200,300);

當然在前面也可以傳入其他指定參數

function moreMoney(name , ...money){  console.log(money);// 小明 [100, 200, 300] } moreMoney('小明',100,200,300);
