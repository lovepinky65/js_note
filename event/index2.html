event - 監聽addEventListener

監聽addEventListener clikc時顯示console.log

addEventListener寫法 addEventListener('選擇事件','代入匿名function',選擇true或false);

false(事件氣泡 event bubbling) - 從指定元素往外找
true(事件捕捉 event capturing) - 從最外層往內找到指定元素

範例

var btn = document.querySelector('.js_click'); btn.addEventListener('click',function(e){  console.log('我用addEventListener點擊事件'); },false);
